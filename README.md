## Setup Instructions
1. Clone repository
2. Run `setup_python_env.bat` (Windows) or `setup_python_env.sh` (Linux/Mac)
3. Configure PostgreSQL database
4. Update `DatabaseConfig.java` with your credentials
5. Run with `mvn javafx:run`
